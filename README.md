## SIHADIR

Website absensi bernama SIHADIR yang digunakan untuk mencatat dan mengolah data kehadiran atau presensi mahasiswa dan dosen dalam kegiatan perkuliahan di kelas.

Aplikasi ini nantinya akan diterapkan untuk menunjang layanan akademik khususnya pemantauan kehadiran dalam proses pembelajaran atau perkuliahan di Prodi Teknik Informatika Politeknik Negeri Pontianak.
